<p><strong>The Backside illuminated BSI CMOS Sensor Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Passive Pixel Sensor CMOS</li><li>Active Pixel Sensor CMOS</li></ul></p>
<p><p>Backside illuminated BSI CMOS sensor market includes two main types, Passive Pixel Sensor CMOS and Active Pixel Sensor CMOS. Passive Pixel Sensor CMOS relies on external circuitry to read and process the pixel output, making it a simpler and more cost-effective option. On the other hand, Active Pixel Sensor CMOS integrates processing circuitry within each pixel, offering faster readout speeds and better noise performance. Both types cater to different market needs, with Passive Pixel Sensor CMOS being suitable for basic applications and Active Pixel Sensor CMOS preferred for more advanced imaging requirements.</p></p>
